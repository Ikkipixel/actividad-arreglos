# Actividad Arreglos en Java

## Objetivo

El objetivo de este proyecto es aplicar los conceptos vistos en la Unidad 1 sobre arreglos unidimensionales y estructuras de datos, utilizando Java.

El programa permite ingresar números, trabajar con arreglos y realizar operaciones básicas con ellos.

---

## Descripción

El programa solicita al usuario ingresar 7 números para dos arreglos diferentes.

Luego de ingresar los datos:

- Se calcula la diferencia entre los valores de cada arreglo.
- El resultado se guarda en un tercer arreglo.
- Se muestran los valores del tercer arreglo.
- Finalmente se calcula el promedio de esos datos.

# ¿Cómo ejecutar el programa?

1. Descargar o clonar el repositorio.
2. Abrir el proyecto en un IDE de Java como IntelliJ IDEA.
3. Ejecutar el archivo.
4. Ingresar los números que el programa solicita.
